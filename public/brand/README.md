# Brand Assets

Drop brand files here.

Recommended layout:

- `public/brand/logo-transparent.svg`
- `public/brand/wordmark-transparent.svg`
- `public/brand/logo-on-background.png`
- `public/brand/wordmark-on-background.png`

If you want additional variants, keep the naming explicit, for example:

- `logo-dark.svg`
- `wordmark-light.svg`
- `logo-on-navy.png`

Files in `public/brand` are served from `/brand/*`.
